This folder contains all the files used for conducting the user study of TransFill.

It contains files
1. user_survey.html //the UI interface containing two selections for each questions. 
2. qualification_test  //the folder contains the code and .xml for qualification test, requiring the SDK installed
3. *.npy //saved results of "binary"--> random order of the options for each question "order"--> overall order or presented samples, *feedback.npy--> the organized user feedbacks generated by analysis.py
4. *full_data.csv, user_feedback.csv //raw data and addresses of samples
5. analysis.py //analysis script of the users' feedbacks
